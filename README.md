# X世代Pokemon傷害計算器
有別於傳統單對單計算器，為無限隻怪同時計算的傷害計算器，另附有HP、速度調整功能(前身為次世代Pokemon傷害計算器)

[https://catho7.github.io/PokemonCalc/](https://catho7.github.io/PokemonCalc/)

## 使用Tips
* 善用抓圖功能保存結果
* 調整的意義在於對戰時，你會知道發生什麼、不會發生什麼，並能從而判斷出對手的道具等
* 未實作功能可使用能力等級暫代(例:風沙時岩系特防x1.5效果->特防能力等級+1、攻擊力x2效果->攻擊能力等級+2)

## 未實裝功能/內容
* 部份圖片
* 部份道具
* 部份特性效果
* 部份招式效果
* 光牆效果
* 部份天氣、場地效果
* 急所傷害計算
* etc
